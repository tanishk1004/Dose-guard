# 💊 DoseGuard — Smart Medication Management

> Never Miss. Never Risk.

DoseGuard is a browser-based medication management app that helps patients track
their daily doses, monitor side effects, and build consistent medication habits —
with zero installation and no login required.

---

## 🚀 Live Demo

Open `index.html` directly in any browser — no server needed.

---

## 🎯 Problem Statement

Medication non-adherence causes over 125,000 deaths and costs $300 billion annually
in the US alone. Most patients forget doses or ignore side effect patterns until it's
too late. DoseGuard gives anyone a zero-friction tool to stay on track.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📋 Medication Scheduler | Add medicines with name, dosage, frequency & time |
| ✅ Dose Tracking | One-tap Taken / Missed buttons per medication |
| 📊 Live Adherence Stats | Real-time taken count, missed count & adherence % |
| 🔥 Streak Counter | Tracks consecutive days of full medication compliance |
| ⚠️ Side Effect Tracker | Logs symptoms with timestamps, warns at 3+ repeats |
| 🔔 Push Notifications | Browser reminders fire at exact scheduled med time |
| 💾 Persistent Storage | All data saved via localStorage — survives page refresh |
| 📱 Responsive UI | Works on desktop and mobile |

---

## 🛠 Tech Stack

- **HTML5** — Structure
- **CSS3** — Styling with animations and gradients
- **Vanilla JavaScript** — All logic, no frameworks
- **Web Notifications API** — Browser push reminders
- **localStorage** — Client-side data persistence
